# Evidence attachments

Store **redacted** milestone review evidence here (or link from your ticket).

**Never commit:** private keys, API secrets, passphrases, raw auth headers, or unredacted responses containing them.

Suggested filenames:

- `v1_00_signoff.md` — reviewer names, date, link to PR, checklist
- `v1_01_approval.md` — resolution-path approval + production-universe caveat
- `v1_03_approval.md` — skeleton / logging / packaging approval
- `v1_02_venue_constraints.md` — observed share vs BUY-notional limits (smoke troubleshooting)
