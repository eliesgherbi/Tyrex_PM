# Scenarios

Maintained folders:
- **`shadow_validation/`** — bundled strategy + risk + **shadow** runtime for smoke and reporting validation. See its `README.md`.
- **`live_validation/`** — bundled strategy + risk + **live** runtime (isolated state). See its `README.md`.

Older named scenario trees were removed; use **`config/runtime/`**, **`config/risk/`** profiles (`*_phaseb_*`), and **`config/strategy/`** for production-shaped pieces.
